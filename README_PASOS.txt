PASOS PARA NAVEGAR ENTRE PÁGINAS (Tecno Galería)
=================================================

1) Copiá estas páginas a tu proyecto:
   - index.html, productos.html, nosotros.html, contacto.html
   - (opcional) promos.html (solo informativa)
   - carpeta css/estiloscolores.css (si ya tenés tu CSS propio, podés ignorar este y mantener el tuyo).

2) Verificá que el menú de TODAS las páginas tenga los enlaces a archivos distintos (ya viene listo).

3) Agregá la página interactiva de promociones:
   - Usá los archivos promociones.html + promociones.js del ZIP corregido (Tecno Galería).
   - Dejalos en la misma carpeta que estas páginas.

4) Subí todo a GitHub y actualizá GitHub Pages.
   URL esperada de promos:
   https://milicarvajal234.github.io/tpo-local-comercial-final/promociones.html

TIP: Si antes usabas anclas (#seccion), reemplazá por páginas reales (index.html, productos.html, etc.).
